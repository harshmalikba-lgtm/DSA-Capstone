# 📘 Data Structures Lab Experiments 

## 🧑‍💻 Overview

This repository contains implementations of core **Data Structures concepts** as part of the **ETCCDS202 Lab Manual**.
The experiments focus on practical understanding of trees, graphs, hashing, and probabilistic data structures using **Python**.

Each program demonstrates both **theoretical concepts** and **real-world applications**, along with clear outputs for verification.

---

## 🎯 Objectives

* Understand fundamental data structures and algorithms
* Implement operations with proper logic and efficiency
* Analyze time and space complexity
* Apply concepts in real-world scenarios (industry relevance)

---

## 📂 Experiments Included

### 🔹 Experiment 20: Binary Search Tree (BST)

* Insert, Search, Inorder Traversal
* Proof of sorted output using inorder traversal

---

### 🔹 Experiment 21: BST Deletion

* Delete nodes (Leaf, One Child, Two Children)
* Maintain BST property after deletion

---

### 🔹 Experiment 22: Heap / Priority Queue

* Min-Heap implementation
* Insert, Peek, Extract operations
* Demonstrates priority-based scheduling

---

### 🔹 Experiment 23: Graph Representation

* Directed weighted graph
* Adjacency list implementation
* Efficient storage and traversal

---

### 🔹 Experiment 24: Breadth-First Search (BFS)

* Level-order traversal using queue
* Shortest path in unweighted graphs

---

### 🔹 Experiment 25: Depth-First Search (DFS)

* Recursive graph traversal
* Backtracking and deep exploration

---

### 🔹 Experiment 26: Hash Table (Separate Chaining)

* Collision handling using linked lists
* Insert, Search, Delete operations

---

### 🔹 Experiment 27: Trie (Prefix Tree)

* Insert, Search, Prefix Matching
* Used in autocomplete systems

---

### 🔹 Experiment 28: Bloom Filter

* Probabilistic data structure
* Memory-efficient membership testing
* Demonstrates false positive concept

---
